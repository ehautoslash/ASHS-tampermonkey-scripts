This script auto-fills the Create Ticket form at "avis.us.abgcustomerservice.com".
Put the confirmation number and last name in the boxes and click the "Autofill Avis Form" button. 

The only piece of information that needs to be manually filled in is the total price from the final receipt.
Because of Avis's poor cookie management, the best way to get that is to right-click "Get Receipt" and open it in incognito/private mode.
Then, you can use the "Copy" button next to the last name and confirmation number boxes to fill in the "Get Receipt" page.

For the attachment, you can use the "View Reservation" link to view the reservation on abglac.com. Take a screenshot and attach it to the ticket.
