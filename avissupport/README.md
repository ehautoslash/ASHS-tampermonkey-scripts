This script auto-fills the Create Ticket form on "avis.us.abgcustomerservice.com".
Enter the last name and confirmation number in the boxes, then click the "Autofill Avis Form" button.
The script takes the information from the box, creates an abglac.com lookup link, and pulls the base rate and total price from the page to fill in the form.

The only piece of information that needs to be manually filled in is the total price from the final receipt.
Because of Avis's poor cookie management, the best way to get that is to right-click "Get Receipt" and open it in incognito/private mode.
Then, you can use the "Copy" button next to the last name and confirmation number boxes to fill in the "Get Receipt" page.

For the attachment, please use the "View Reservation" button to view the reservation on abglac.com. Take a screenshot and attach it to the ticket.
